Chức năng phân tích bao gồm:

  • Behavior Analysis: Phân tích runtime behaviors từ MongoDB
  • Pattern Detection: Phát hiện các attack patterns (credential theft, session hijacking, etc.)
  • Anomaly Detection: Phát hiện behaviors bất thường
  • Risk Scoring: Tính điểm rủi ro theo chuẩn Google (0-100)
  • Static Analysis: Phân tích manifest + JS code (nếu bật)
